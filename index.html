Лендинг для свадебной флористики, который аккуратно презентует студию агентствам, объясняет формат сотрудничества и ведет к личному обсуждению проекта.

01 / Задачи

Что должен решить лендинг

Задача страницы — не просто красиво показать цветы, а выстроить понятную B2B-презентацию для свадебных агентств: с акцентом на сервис, процесс и надежность.

{ 01 }

Презентовать BY ME как партнера

Показать студию не только как флористов, а как команду, с которой агентству спокойно вести свадебный проект.

{ 02 }

Объяснить полный цикл услуг

Разложить, за какие зоны отвечает команда: букет, церемония, банкет, welcome-зоны, инсталляции, логистика и монтаж.

{ 03 }

Довести до обращения

Собрать страницу так, чтобы после кейсов, преимуществ и процесса пользователю было естественно перейти к обсуждению проекта.

02 / Структура

Предлагаемая структура лендинга

В основе — уже подготовленная логика страницы. Она ведет пользователя от первого впечатления к пониманию услуг, формата работы и контакту.

{ 01 }

Header / навигация

Логотип, ссылки на ключевые разделы, кнопка для связи.

{ 02 }

Первый экран

Главный оффер, краткое описание и визуальная подача бренда.

{ 03 }

О BY ME

Короткое знакомство со студией, подходом и ощущением работы.

{ 04 }

Что предлагаем

Список услуг: букет, церемония, банкет, welcome, инсталляции, сопровождение.

{ 05 }

Особая услуга

Примерка букета с образом невесты как дополнительный акцент и отличие.

{ 06 }

Кейсы

Примеры оформлений, задачи проектов, теги, фотографии и краткие описания.

{ 07 }

Партнерство

Аргументы, почему агентству будет спокойно работать с BY ME.

{ 08 }

Процесс работы

Последовательность: погружение, решение, согласование, подготовка, реализация.

{ 09 }

Финальный CTA / контакты

Приглашение обсудить проект, мессенджер, почта, телефон и социальные сети.

03 / Этапы

Этапы работ

Работа идет последовательно: сначала фиксируем смысл и структуру, затем визуал, сборку на Tilda и итоговую подготовку страницы.

этап 01

Анализ и структура

  • изучение сферы и конкурентов;
  • анализ визуальной подачи и структуры;
  • определение логики страницы и CTA.
Результат: понятная структура лендинга и список смысловых блоков.
этап 02

Прототип

  • расположение блоков и основных элементов;
  • распределение текстов, фото и кнопок;
  • проверка логики страницы до дизайна.
Результат: черновой скелет страницы, по которому можно дорабатывать дизайн.
этап 03

Дизайн-доработка

  • работа с композицией, сеткой и ритмом;
  • оформление карточек, кейсов и акцентов;
  • подготовка визуальной логики для Tilda.
Результат: цельное визуальное направление будущего лендинга.
этап 04

Верстка на Tilda

  • сборка desktop-версии;
  • мобильная адаптация блоков;
  • кнопки, ссылки, якоря, базовая анимация;
  • проверка отображения и визуальная полировка.
Результат: рабочая страница на Tilda, готовая к финальному согласованию.
этап 05

Финализация

  • внесение согласованных правок;
  • проверка ссылок, кнопок и адаптива;
  • подготовка страницы к публикации.
Результат: финальная версия лендинга для публикации.

Основные условия сотрудничества, которые помогают заранее зафиксировать формат, сроки и дополнительные расходы.

Я оформлена официально и работаю по договору как самозанятая. Все условия сотрудничества закрепляем документально.

Оплата Tilda Publishing

{ 02 }

Аккаунт на Tilda на один сайт стоит 750 ₽/мес или 6000 ₽/год. Стоимость подписки не входит в стоимость услуг. Подробнее о тарифах Tilda.

Правки

{ 03 }

В оба тарифа входят 2 итерации правок на каждый этап. Если этого количества будет недостаточно, дополнительные правки оплачиваются отдельно — 2000 ₽/час.

Сроки работ

{ 04 }

Сроки работ четкие и будут прописаны в договоре. При начале сотрудничества будет выслан график работ, которого мы будем придерживаться.

Буду рада сотрудничать

Напишите, какой из тарифов вам ближе, мы зафиксируем условия и перейдём к графику работ. Если ни один из форматов не подходит, то я готова обсудить пожелания и подобрать подходящее решение.
